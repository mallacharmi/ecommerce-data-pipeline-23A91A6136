# Dashboard User Guide

## Accessing the Dashboard

Power BI:
Open dashboards/powerbi/ecommerce_analytics.pbix using Power BI Desktop

## Dashboard Pages

### Page 1: Executive Summary

Purpose: High-level overview of business performance

Key Metrics:

- Total Revenue
- Total Transactions
- Average Order Value
- Profit Margin

Visuals:

- Monthly revenue trend
- Top categories
- Payment method distribution
- State-wise revenue (bar chart)

### Page 2: Product Performance

Purpose: Analyze product-level performance

Key Insights:

- Top 10 products generate majority of revenue
- High-priced products yield higher profit margins

### Page 3: Customer Analytics

Purpose: Understand customer behavior

### Page 4: Trends & Geography

Purpose: Analyze time and location patterns

Key Insights:

- Certain states dominate revenue contribution
- Weekday sales outperform weekends
